## 🌟 Overview
Stand out to recruiters with a clean, code-based resume. This responsive HTML/CSS template is beautifully structured, completely print-ready (exports perfectly to PDF), and easy to host on GitHub Pages.

## ✨ Key Features
- 🖨️ **Print Optimized**: Specifically styled for perfect A4 PDF export.
- 📱 **Fully Responsive**: Looks great on mobile, tablet, and desktop.
- 🎨 **Clean Typography**: Uses modern, highly readable sans-serif fonts.

## 🚀 Getting Started

### Prerequisites
Make sure you have `HTML_CSS` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/minimalist-html-resume.git
cd minimalist-html-resume
No build step required.
```

### Usage
```bash
Open index.html in your browser and edit the text!
```
